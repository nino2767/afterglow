상위기획 결과

한 줄 정의
AFTERGLOW는 전시 IP의 세계관을 확장해 '두 번째 작품(스핀오프)'으로 만들어내는 서비스다. 본전시를 봤든 안 봤든 그 자체로 성립하는 독립된 세계관 경험이며, 관람의 여운과 개인 데이터는 이 경험을 더 깊게 만드는 레이어다.

핵심 전환: "본전시의 연장 장치" → "세계관을 공유하는 독립된 두 번째 작품"

핵심가치 키워드 본전시 세계관의 확장(독립된 두 번째 작품) · 정보가 아닌 경험 · 여운·개인화는 정체성이 아닌 '강화 레이어'
우선순위 명확화 · 2026.07.16 1순위 정체성 = 본전시의 새로운 세계관(독립 작품). 여운·개인화는 이를 대체하지 않는 강화 레이어다. 7장(Phase 1 스핀오프는 본전시 데이터 없이 성립, 개인화는 Phase 3 프리미엄)과 언어를 일치시킴. 경쟁력의 무게가 개인화 대신 '새 세계관을 빠르고 일관되게 찍어내는 AI 엔진'으로 이동함에 유의.

원점 (동기)

  • 영화/예능의 '스핀오프' 개념을 전시로 — 본편이 끝나도 세계관이 이어진다
  • 개인 경험: 전시 후 작가·작품을 검색해도 정보 파편만 있고 허전했음 (이해+감동+연결 모두 미해소)
  • 시장 공백(⚠️ 재검증 필요): "전시/미술은 IT 낙후"라는 초기 인상은 수정 필요. 관람 중 가이드 영역은 이미 경쟁 존재 — 오프라인 도슨트 보편화, 국립중앙박물관은 IoT 비콘 기반 위치 자동안내(5개 언어)에 이어 2025.9 AI 검색·지능형 큐레이션 플랫폼 공개, 네이버는 바이브(VIBE) 오디오 탭에서 전시 도슨트 콘텐츠 운영(NCT 127 도슨트, 마티스전 등 — 전시 종료 후에도 콘텐츠 제공), 오디오클립은 2025.12 서비스 종료 확정. 다만 이들은 모두 관람 중 서비스이며, 관람 후 세계관 연장(스핀오프) 영역은 여전히 공백 → AFTERGLOW의 공백 정의를 '전시 IT 전반'이 아니라 '관람 후 구간'으로 좁혀야 함. 상세는 별도 문서 AFTERGLOW_시장조사리포트_v1.0.md 참조

핵심 동작

전시 IP(작품, 작가 세계관)를 재료로, AI가 새로운 세계관 경험 — 이어지는 대화, 미니 스핀오프 전시, 커스텀 굿즈 — 을 독립된 '두 번째 작품'으로 만들어낸다. 본전시를 보지 않아도 성립하며(Phase 1), 관람객의 사진·대화 데이터가 붙으면 그 경험이 개인화되어 더 깊어진다(Phase 3 프리미엄).

B2B의 위상 본질이 아니라 전달 경로. 세계관의 주인이 기획사이므로 그들을 통해 관람객에게 닿는다.

무엇을 검증하는가

"전시가 끝난 후에도 관람객은 그 세계관에 머물고 싶어하며, 스핀오프 경험(이어지는 대화 + 개인 아카이브 + 미니 오프라인 체험)이 '검색하고 허전한' 경험을 대체할 수 있다."

검증 플로우

관람 중사진 촬영 → AI 해설·대화
퇴장 시점자동 아카이브 → 2경로 초대
온라인 경로재접속·세계관 대화·공유
오프라인 경로미니 스핀오프 전시
  1. 관람 중: 작품 사진 촬영이 기본 진입점 — 작품을 찍으면 AI가 사진을 인식해 설명 제공 + AI 큐레이터 대화 시작 (사진·감정 키워드가 개인 데이터로 축적). QR은 보조 진입 수단(선택) — QR이 없는 전시장도 많음. 촬영 불가 전시는 작품 리스트에서 직접 선택하는 폴백 구조
  2. 퇴장 시점: 사진+대화가 자동으로 나만의 아카이브(리뷰 초안)로 생성 → 두 경로를 독립적으로 초대 (① 온라인 스핀오프 ② 미니 스핀오프 전시) — 각각 분리 측정
  3. [오프라인 경로] 미니 스핀오프 전시 (전시장 내/인근 1개 공간): 굿즈숍이 아니라 '또 하나의 작은 전시'를 보는 경험 — 본전시 세계관을 가져와 다른 이야기·연출로 구성한 새 장면을 관람. 예능 스핀오프 문법을 따름 (아는형님→방과후교실, 나혼자산다→여은파: 원작의 세계관·인물로 다른 판을 벌이는 독립 콘텐츠). 내 관람 데이터는 연출에 반영 (예: 내 감정 키워드가 공간의 빛·사운드·장면으로). 굿즈는 관람 동선 출구의 기념품 — 본체가 아님
  4. [온라인 경로] 관람 후 (D+1~7): 웹앱 재접속 → 세계관 대화 계속, 아카이브 확인·공유

온라인/오프라인 분리 검증 설계

두 경로는 서로 독립 — 부스를 안 가도 온라인 스핀오프는 온전히 즐길 수 있고, 그 역도 성립. 결과 조합별 판정:

온라인 재접속부스 방문해석 → 다음 방향
↑↑풀 스핀오프 검증 → 정식 팝업 확장 + 디지털 병행
↑↓머무름 욕구는 디지털로 충족 → 디지털 스핀오프 중심, 팝업은 선택 상품
↓↑오프라인 체험이 본체 → 팝업 중심(v3.0 원형에 근접), 웹은 브릿지로 축소
↓↓본질 가설 재검토 (상위기획 복귀)

기능 범위 IN / OUT

In Scope

MVP에 포함

  • 웹앱형 AI 큐레이터 (핵심 작품 3~5개 대화 해설) — 진입은 사진 인식 기반(QR 보조, 촬영 불가 시 작품 선택 폴백)
  • 사진 촬영 → 자동 아카이브/리뷰 초안 생성 (신규)
  • 관람 후에도 이어지는 세계관 대화 (작가·작품 심층 대화) — 본질 직결
  • 미니 스핀오프 전시 1개 공간 (세계관 기반 새 장면 + 출구 굿즈 1~2종)
  • 상설 데모 전시 1개 (v3.0 '심해의 환상' 실구현) — 라인업·세일즈 데모·파이프라인 테스트베드
  • 내부용 AI 기획 어시스턴트 (부스 컨셉·굿즈 생성용, 비노출)
Out of Scope

MVP에서 제외

  • 정식 팝업 공간 운영 대행 (21일 연장 운영 등) → 검증 후
  • 네이티브 앱, 자체 결제, 실시간 도면 생성, 다국어 (v3.0 OUT 유지)
  • 전시 검색 기능
  • 기획사용 대시보드 (파일럿에선 수동 리포트로 대체)
