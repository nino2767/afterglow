# AFTERGLOW 시장 조사 리포트 v1.0

작성일: 2026.07.11 | 목적: "시장 공백" 가설 재검증 — 공백은 '전시 IT 전반'인가, '관람 후 구간'인가

---

## 요약 (3줄)

1. **관람 중 가이드는 레드오션** — 오프라인 도슨트 보편화 + 국공립 기관의 AI 도슨트 자체 도입 + 큐피커·Smartify 등 앱 경쟁 존재. "전시/미술 IT 낙후" 가설은 기각.
2. **관람 후 '기록'은 부분 경쟁, '세계관 연장'은 공백** — 아카이빙 앱은 존재하지만 모두 수동 기록 도구. 전시 IP를 두 번째 작품(스핀오프)으로 잇는 서비스는 국내외 미발견.
3. **전시 IP 2차 수익화의 시장성은 입증됨** — 아르떼뮤지엄 실적(상품판매 급증, 해외 월 수백만 달러)이 방증. 단, 자사 IP 모델뿐이며 '외부 IP 대행'은 공백.

## 2. 관람 후 기록/아카이브 영역 — 부분 경쟁 🟡

| 플레이어 | 내용 | 한계 (= AFTERGLOW 기회) |
|---|---|---|
| 아트맵 | 취향 기반 전시 큐레이션, '다녀왔어요' 컬렉션, 취향 분석 | 정보 플랫폼 + 수동 기록. 세계관 경험 없음 |
| 아트파인더 | '체크인'으로 관람 서사 아카이빙, 감정·사진·후기 기록 | 사용자가 직접 써야 함 (노력 필요) |
| Archive 앱 | 전시 기록·감정 보관 | 동일 — 기록 도구 |

**소결**: '관람 후'를 다루는 앱은 있으나 전부 **사용자가 노력해서 남기는 기록 도구**. AFTERGLOW의 차별점 두 가지가 유효 — ① 사진·대화가 **자동으로** 아카이브가 됨(Zero-effort), ② 기록을 넘어 **세계관 경험이 계속됨**(스핀오프 대화·콘텐츠·굿즈·부스). 후자는 조사 범위 내 유사 사례 없음. **공백 확인.** 🟢

## 3. 스핀오프/팝업/IP 2차 수익화 영역 — 시장성 입증, 대행 모델은 공백 🟢

| 플레이어 | 내용 | AFTERGLOW 관점 |
|---|---|---|

| 프로젝트렌트 | 성수동 중심 팝업 전용 공간 8곳, 기획~운영 대행. "매장을 미디어로" | 브랜드 마케팅 팝업 중심, 전시 IP 특화 아님. 수동 기획 |
| 팝플리·팝가 | 팝업 정보 플랫폼 (일정·예약·후기) | 인프라/채널로 활용 가능 |
| 마플/마플샵 (2026.07 확인) | POD 재고 없는 주문제작, 디자인 1개로 1,500+ 상품 카테고리. **마플샵 플러스** = B2B 인프라 솔루션 (사이트 구축+주문·생산·배송·CS). 라인프렌즈가 이걸로 'IP 커스터마이징 플랫폼(1개부터 제작)' 구축 — AFTERGLOW 출구 굿즈와 동일한 유스케이스 | MVP 출구 굿즈 제작 경로로 유력. 공개 API 문서는 없음 → B2B 제휴 직접 문의 필요 |

**소결**: '전시 IP 스핀오프 대행'이라는 포지션은 여전히 비어 있음. 아르떼뮤지엄이 증명한 수익성을 외부 IP에 서비스로 제공하는 것이 AFTERGLOW의 자리.

## 4. 종합 시사점

1. **공백의 재정의 확정**: "전시 IT 낙후" ❌ → **"관람 후 세계관 연장 구간의 공백"** ✅. v4.0의 본질 재정의와 일치.
2. **모듈 B(스핀오프) 메인 전략 타당**: 관람 중 가이드는 공공·앱·오프라인 3면 경쟁, 관람 후 스핀오프는 무주공산.
3. **사진 인식 진입은 검증된 기술**: Smartify가 글로벌 스케일로 증명 — 기술 리스크보다 콘텐츠(세계관) 품질이 승부처.
4. **⚠️ 피플리 이중 관계**: 1순위 파트너 피플리가 큐피커 운영사. 모듈 A로 접근하면 경쟁 제안이 됨 — **"큐피커가 못 하는 관람 후 구간(스핀오프)을 붙여주는 파트너"**로 포지셔닝해야 함. 오히려 큐피커의 관람 데이터와 연동하는 협업 그림도 가능.
5. **B2C 셀럽 도슨트 참고**: VIBE의 NCT 127 사례 — 스핀오프 콘텐츠에 '작가/셀럽 보이스' 결합은 검증된 흥행 장치.
