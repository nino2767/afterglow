# 8. 오프라인 터치포인트와 후속 연결 구조

| 위치 | 관람객 행동 | 서비스 연결 | 후속 스핀오프 연결 |
| --- | --- | --- | --- |
| 입구 | QR 스캔 | 관람 모드 시작 | 관람 세션 생성 |
| 작품 옆 | QR 스캔 | 작품 상세 확인 | 작품/작가 기반 추천 |
| 섹션 입구/출구 | 섹션 QR 스캔 | 주제 반응 수집 | 서사/테마 기반 추천 |
| 포토존 | QR 스캔/공유 | 감상 카드 저장 | 팝업/굿즈 추천 |
| 굿즈샵 | 저장 작품 확인 | 작품 기반 MD 노출 | 굿즈 구매/출시 알림 |
| 출구 | 관람 완료 QR | 리포트 생성 | 스핀오프 추천 |
| 관람 후 모바일 | 리포트 확인 | 후속 콘텐츠 제공 | 알림/예매 전환 |

# 9. 데이터 수집 및 스핀오프 연결 로직

| 관람객 반응 | 해석 | 연결되는 스핀오프 |
| --- | --- | --- |

| 관계성 콘텐츠 클릭 높음 | 인물 간 관계 관심 | 관계 서사형 전시 |
| 감정 태그 반복 선택 | 정서적 몰입 | 감정 테마형 전시 |
| 포토존 저장/공유 높음 | 시각적 확산성 | 몰입형 팝업 |
| 굿즈 보기 클릭 | 상품화 관심 | 한정 굿즈, 아트프린트 |
| 관련 작품 클릭 높음 | 확장 탐색 | 동시대 작가 연결 전시 |
| 리포트 재방문 높음 | 전시 후 지속 관심 | 온라인 콘텐츠 구독 |
| 스핀오프 관심 등록 | 후속 경험 의향 | 기획 검증 |
| 알림 신청 | 재접촉 가능성 | 사전예약 전환 |
| 예매 | 구매 의사 | 사업화 가능성 |
