01개요 및 기획 목적

B2B 기획 관리 플랫폼
본 시스템은 B2B 고객사(기획사/작가)가 전시 자산을 등록하면, AI 엔진이 이를 기반으로 독립된 '두 번째 작품(스핀오프 전시)' 콘텐츠를 자동 빌딩 및 운영할 수 있도록 지원하는 백오피스입니다.
B2B 셀링포인트: 기획 대행 비용 절감 · 콘텐츠 생성 프로세스 자동화 · 통합 B2B 성과 리포팅

B2C의 독립 스핀오프 유저뷰가 원활하게 작동하기 위해서는, 이들의 뼈대가 되는 전시 IP 데이터셋을 신속하고 효율적으로 생산·검증하는 어드민의 '기획 어시스턴트' 엔진이 필수적으로 수반되어야 합니다.

03핵심 기획 정책

AI 콘텐츠 승인 프로세스 (RAG 파이프라인)

  1. Draft 상태 (자산 등록): 전시 기획사 담당자가 도슨트 텍스트 및 세계관 배경자료를 등록함.
  2. AI Generating 상태: AI RAG 엔진이 정보를 토대로 5대 공통 기능 모듈 규격에 맞게 씬 스토리, 대화 컨셉, 굿즈 추천을 자동 설계함.
  3. Review 상태: 어드민 페이지 시뮬레이션 창을 통해 씬 콘텐츠를 모니터링하고 수정사항을 수동으로 교정함.
  4. Published 상태: 전시 기획사 컨펌 완료 후 스핀오프 고유 URL 및 QR을 오픈함.

04코어 데이터 모델

S님이 공통 기획/개발을 진행할 핵심 코어 스키마 정의안입니다.

Exhibition IP Schema (전시 자산)

{
  "exhibition_id": "string (UUID)",
  "title": "string",
  "artist": {
    "name": "string",
    "profile_desc": "string"
  },
  "worldview_context": "string (RAG 학습용 장문 텍스트)",
  "artworks": [
    {
      "artwork_id": "string",
      "title": "string",
      "vision_reference_image_url": "string (URL)",
      "description": "string",
      "audio_script": "string"
    }
  ]
}

Spinoff Scene Schema (스핀오프 씬)

{
  "spinoff_id": "string (UUID)",
  "exhibition_id": "string (Exhibition UUID)",
  "persona": {
    "name": "string",
    "tone_type": "string (friendly | academic | emotional)",
    "system_instruction": "string (AI 프롬프트)"
  },
  "scenes": [
    {
      "scene_number": "integer",
      "title": "string",
      "story_content": "string (외전 텍스트)",
      "bg_ambient_music_url": "string",
      "visual_preset_color": "string (Hex code)"
    }
  ]
}

User Experience Session Schema (유저 세션)

{
  "session_id": "string (UUID)",
  "is_visitor_of_main_exhibition": "boolean",
  "linked_main_session_id": "string (Optional, 본전시 연동 시)",
  "user_sentiment_codes": ["string (감정 코드 리스트)"],
  "spinoff_status": {
    "current_scene": "integer",
    "chat_turn_count": "integer",
    "is_completed": "boolean"
  },
  "conversions": {
    "card_shared": "boolean",
    "goods_clicked": "boolean",
    "main_ticket_clicked": "boolean"
  }
}
