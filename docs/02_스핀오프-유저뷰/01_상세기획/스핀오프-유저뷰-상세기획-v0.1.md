# 스핀오프 유저뷰 상세기획서 v0.1

> 작성일: 2026.07.16 | 상태: 📋 초안 확정
> 근거: 방향성 재정의 v4.0 draft, 스핀오프 유저뷰 화면플로우 v0.1, 스핀오프 유저뷰 IA v1.0
> 담당: **Sol (Spinoff)**

본 문서는 스핀오프 전시 유저 서비스(S-1 ~ S4) 및 신규 본전시 상세/예매 화면(S1.5)에 대한 기능적, 시각적 동작 규칙과 데이터 상태 기획을 고정합니다.

---

## [공통 디자인 가이드라인]

* **테마**: 다크 모드 중심의 몰입형 테마 (`background: #0D0D0F`, `color: #FAFAF9`)
* **포인트 컬러**:
  * 스핀오프 키 포인트: 글로우 블루 (`var(--glow)` = `#4FD8EB` / `var(--glow-dim)` = `#2A7F96`)
  * 본전시 키 포인트: 버건디 (`var(--accent)` = `#8B2E4A`)
* **서체**: 제목 폰트 (`Playfair Display`, 세련된 세리프) + 본문 폰트 (`Inter` / `Pretendard`, 고대비 산세리프)
* **뷰포트**: 모바일 뷰포트 기준 (`max-width: 430px` 센터링 레이아웃)

## 1. S-1. 서비스 홈 (Service Home)

* **경로**: `/spinoff/home`
* **역할**: 브랜드 메인 첫 화면이자 상영작 라인업 소개 및 고객사 미팅용 쇼케이스 랜딩.

### 1-1. 화면 구성 및 UI 요소
1. **브랜드 헤더**: `AFTERGLOW` 로고 (글로우 텍스트 변주)
2. **섹션 타이틀**: "지금 상영 중인 스핀오프"
3. **상영작 라인업 리스트 (카드 형태)**:
   * **[카드 1] 데모 상설 전시**: 《심해의 환상 : 두 번째 물결》
     * 뱃지: `DEMO 상설`
     * 이미지/그라디언트 배경: 심해 느낌의 블루-블랙 웨이브 패턴
     * 정보: "본전시 〈심해의 환상〉 세계관 · 온라인+성수"
     * 액션: 클릭 시 S0 포스터 화면(`/spinoff/abyss-wave`)으로 이동
   * **[카드 2] 파트너 전시 스핀오프 (가상)**:
     * 뱃지: `Coming Soon`
     * 정보: "다음 세계관이 준비 중입니다."
     * 상태: 비활성 처리
4. **B2B 소개 안내 (하단 푸터 영역)**:
   * "기획사 제안 문의" 링크 배치 (B2C 몰입감을 해치지 않도록 작고 정적으로 텍스트 노출)

## 2. S0. 발견 (디지털 포스터)

* **경로**: `/spinoff/[slug]` (예: `/spinoff/abyss-wave`)
* **역할**: 개별 스핀오프 전시의 첫 인상과 컨셉 키비주얼을 보여주는 모바일 디지털 포스터.

### 2-1. 화면 구성 및 UI 요소
1. **컨셉 배경 (히어로 키비주얼)**: 화면 전체에 그라디언트 및 앰비언트 웹 애니메이션(radial-gradient 움직임) 적용
2. **카테고리 태그**: `SPIN-OFF EXHIBITION`
3. **전시 타이틀**: 《심해의 환상 : 두 번째 물결》
4. **전시 서브카피**: "전시가 끝난 곳에서, 이야기는 이어집니다. 당신이 미처 만나지 못한 심해의 네 장면."
5. **핵심 버튼**: `[입장하기]` (글로우 테두리 및 반짝임 효과)
6. **출처 푸터**: "원작 전시 · 심해의 환상 (2026.5–7) | 본전시를 안 봤어도 입장할 수 있어요"

## 3. S1. 입장 (온보딩)

* **경로**: `/spinoff/[slug]/onboarding`
* **역할**: 세계관에 진입하기 위한 초단기 분기 수집 및 감정 키워드 시드 설정 화면.

### 3-1. 화면 구성 및 UI 요소
1. **타이틀**: "본전시 〈심해의 환상〉, 보셨나요?"
2. **분기 카드 1: "아니요, 처음이에요" (미관람자)**
   * 취향 및 감정 키워드 선택 칩 배치 (`고요함`, `몽환`, `압도`, `기괴함`, `따뜻함` 등)
   * **[본전시 정보 링크]**: "원작이 궁금하다면? 본전시 정보 보기 ↗" (텍스트 링크)
     * 클릭 시 `/exhibition/abyss`로 연결
3. **분기 카드 2: "네, 봤어요" (관람 완료자)**
   * "기억에 남는 장면 하나를 고르면 연출에 반영돼요" 설명
   * 장면 선택 목록 (1구역 윤슬, 2구역 심연의 호흡, 3구역 기억의 잔광 등)
4. **데이터 수집 동의 체크라인**:
   * "관람 데이터 수집 동의 (감정 키워드·대화, 아카이브 생성에 사용) · 로그인 없음"
5. **하단 시작 버튼**: `[관람 시작하기]`
   * 동의 체크 및 분기 카드가 하나 이상 선택된 상태일 때만 활성화

## 4. S1.5. 본전시 상세 정보 (Exhibition Info)

* **경로**: `/exhibition/[slug]` (예: `/exhibition/abyss`)
* **역할**: 본전시를 관람하지 않은 유저에게 본전시 홍보 및 예매 전환을 이끄는 전용 스크린.

### 4-1. 화면 구성 및 UI 요소
1. **상단 네비게이션**: 뒤로가기 버튼 (이전 스핀오프 단계로 복귀)
2. **전시 히어로 배너**: 버건디 그라디언트 및 메인 포스터 비주얼
3. **전시 타이틀**: 몰입형 미디어아트 《빛의 심연 (Abyss of Light)》
4. **상세 소개**:
   * "인간의 고독과 안식을 일렁이는 미디어아트로 표현한 몰입형 전시입니다. AI 가이드 음성과 함께 감상해보세요."
   * 전시 일정: 상시 운영 (10:00 ~ 20:00)
   * 장소: 아트홀 서울 지하 1층 특별실
5. **예매하기 버튼**:
   * `[본전시 예매하기 (결제)]` 클릭 시 S1.5.1 본전시 예매/결제 화면으로 이동.

---

## 4.5. S1.5.1. 본전시 예매/결제 (Booking & Payment)

* **경로**: `/exhibition/[slug]/booking`
* **역할**: 실제 결제 모사를 통해 티켓 매수 및 결제 수단을 고르는 예약 입력 화면.

### 4.5-1. 화면 구성 및 UI 요소
1. **예약자 정보 폼**: 이름, 이메일 주소, 연락처 (기본값 제공, 수정 가능).
2. **매수 선택**: 일반 티켓 (1인 15,000원) 매수 조절 (1~5매 제한).
3. **결제 수단**: 신용카드 및 카카오페이 선택 버튼 제공.
4. **결제 진행**: `[가상 결제하기]` 클릭 시 1.5초 로딩 상태(스피너)를 거친 후 S1.5.2 예매완료로 전환.

---

## 4.6. S1.5.2. 본전시 예매완료 (Booking Success)

* **경로**: `/exhibition/[slug]/success`
* **역할**: 예매 완료 피드백 제공 및 모바일 pass 바코드/QR 발급 상태 연출.

### 4.6-1. 화면 구성 및 UI 요소
1. **완료 상태 인디케이터**: 체크 애니메이션 및 축하 문구.
2. **모바일 Pass 카드**: 예약자명, 구매 매수, 고유 티켓 번호, 바코드 일러스트레이션.
3. **상태 활성화 로직**: 예매 완료와 동시에 브라우저 `localStorage`에 본전시 세션(`afterglow_session`)과 계정 정보(`afterglow_account`)를 자동 등록하여, 전체 앱에서 활성 관람자로 상태 전이시킴.
4. **액션 버튼**:
   * `[본전시 작품목록 보기]`: `/artwork` (본전시 작품목록) 화면으로 바로 진입.
   * `[메인 대시보드로]`: 메인 계정 홈(`/home`)으로 이동.

---

## 4.7. S1.6. 본전시 완료 및 초대권 발급 (Exhibition Completed)

* **경로**: `/exhibition/[slug]/completed`
* **역할**: 본전시 관람을 완수했음을 축하하고, 스핀오프 전시 초대 티켓과 연결 단추를 강조 제공하여 스핀오프 재진입을 유도하는 화면 (Flow B).

### 4.7-1. 화면 구성 및 UI 요소
1. **완료 축하 멘트**: "관람이 완료되었습니다. 마음속에 새겨진 단서로 빚어낸 스핀오프 특별 초대장이 도착했습니다."
2. **스핀오프 특별 초대권Pass**:
   * 뱃지: `초대 전용 관람권`
   * 타이틀: "어비스 티 라운지 (Abyss Tea Lounge)"
   * 텍스트: "전시가 끝난 곳에서, 이야기는 이어집니다. 당신이 미처 만나지 못한 심해의 네 장면."
   * 디자인: 시안색(글로우) 글로잉 보더라인 테마, 초대 바코드 일러스트.
3. **스핀오프 연결 단추**: `[스핀오프 전시 보러가기 ↗]` 클릭 시 `/spinoff/abyss-wave`로 이동하여 스핀오프 발견/포스터 화면으로 자연스러운 관람 연결을 유도.

## 5. S2. 관람 (씬 뷰어)

* **경로**: `/spinoff/[slug]/viewer`
* **역할**: 스핀오프의 신규 예술 장면을 감상하며 AI 큐레이터와 대화하는 몰입형 코어 뷰어.

### 5-1. 화면 구성 및 UI 요소
1. **뷰 모드 토글 (공통 모듈 연계)**:
   * **[몰입형 뷰 (Swipe Mode)]**: 세로 스와이프 3개 장면 뷰어, AI 대화창 오버레이 버블.
   * **[목록형 뷰 (List Mode)]**:
     * 작품 목록 (`ModularArtworkList`) 렌더링.
     * 카드 클릭 시 작품 상세설명 패널 (`ModularDocentPanel`) 팝업 오픈.
     * 상세 설명 패널 내 `[AI 큐레이터와 대화 나누기]` 클릭 시 챗봇 화면(`/spinoff/[slug]/curator`)으로 연동.
2. **장면 카드 영역 (세로 스와이프)**:
   * 이미지 영역: 움직이는 빛과 어두운 바다 느낌의 추상 비주얼
   * 장면 타이틀: (예: `빛이 닿지 않는 정원`)
   * 장면 나레이션: "당신이 '고요함'을 골랐기에, 이 장면은 조금 더 느리게 숨 쉽니다." (S1 키워드 선택값이 텍스트에 삽입되어 개인화 연출)
3. **AI 대화창 (오버레이 버블)**:
   * "심해의 안내자: 이 정원의 빛은 본전시 마지막 방에서 흘러나온 것들이에요. 무엇이 궁금하세요?"
   * 대화 입력창 + 카메라 아이콘 버튼
     * 카메라 버튼 클릭 시: (가상 카메라 셔터 효과음 또는 모사 다이얼로그 노출)
4. **하단 이동 버튼**: 장면 탐색 완료 후 하단 `[여정 기록하고 아카이브 받기]` 버튼 활성화 (목록형 뷰에서는 아카이브 받기로 즉시 전환 가능)

## 6. S3. 기념 (나의 아카이브)

* **경로**: `/spinoff/[slug]/archive`
* **역할**: 관람 결과물인 개인화 리포트를 확인하고 공유 및 굿즈 연계, 그리고 최종 본전시 예매 전환을 완성하는 화면.

### 6-1. 화면 구성 및 UI 요소
1. **아카이브 헤더**: "당신의 심해는 '고요한 압도'였어요" (선택 키워드의 조합 결과 타이틀화)
2. **모자이크 카드**: 감상한 장면들의 이미지가 모자이크 콜라주 형태로 배열
3. **AI 대화 발췌구**: "빛이 가라앉는 게 아니라, 어둠이 떠오르는 거군요" — 당신과 안내자의 대화 중 발췌
4. **기본 액션 버튼**:
   * `[공유 카드 만들기]`
   * `[이 순간, 굿즈로 간직하기]` (마플 POD 연계 안내 다이얼로그)
5. **[조건부 CTA 연계 버튼 (중요)]**:
   * **유저가 S1 온보딩에서 "아니오"를 선택한 경우에만** 하단에 강조된 스타일의 다음 버튼이 노출됩니다.
   * **버튼 문구**: `[본전시 〈심해의 환상〉 보러가기 ↗]` (글로우 컬러 포인트 스타일)
   * **동작**: 클릭 시 `/exhibition/abyss` (본전시 상세 화면)로 리다이렉트
6. **푸터 버튼**: `[여정 완료하고 홈으로 가기]` -> 클릭 시 메인 대시보드인 `/home`으로 이동

## 7. S4. 여운 (재방문)

* **경로**: `/spinoff/[slug]/retention`
* **역할**: 퇴장 이후 관람객의 지속 방문을 유인하기 위해 D+2 리마인드와 추가 콘텐츠를 제공하는 화면.

### 7-1. 화면 구성 및 UI 요소
1. **알림 모사 헤더**: "AFTERGLOW 알림 · D+2"
2. **추가 콘텐츠**: "🌊 다섯 번째 장면이 열렸어요: 깊어지는 잔상"
3. **AI 대화창**:
   * "심해의 안내자: 다시 오셨네요. 그날 가장 오래 머무셨던 정원에, 그 후 무슨 일이 있었는지 보여드릴까요?"
   * 유저의 대화 입력창 및 과거 대화 아카이브 재로딩
