# 스핀오프 유저뷰 정보 구조 (IA) 정의서 v1.0

> 작성일: 2026.07.16 | 상태: 📋 초안 확정
> 근거: 방향성 재정의 v4.0 draft, 스핀오프 유저뷰 화면플로우 v0.1, 유저뷰 모듈분담 및 Git워크플로우 v1.0
> 담당: **Sol (Spinoff)**

본 문서는 스핀오프 전시 유저뷰(S-1 ~ S4)의 독립화 개편에 맞춰 전체 사이트맵, 라우팅 구조, 유저 분기 및 상태값 흐름을 정의합니다. 특히 본전시 미관람자를 대상으로 하는 **본전시 상세 정보 및 예매 화면(S1.5)**의 추가와 연동 규격을 고정합니다.

---

## 1. 3가지 진입 및 라우팅 모드

스핀오프 서비스 개편으로 인해 기존의 본전시 하위 팝업 개념에서 **독립된 두 번째 작품**으로 격상됨에 따라 다음과 같이 라우팅 모드가 분류됩니다.

| 모드 | 진입 경로 | 특징 | 관련 화면 |
|---|---|---|---|
| **스핀오프 서비스 홈** | 직접 검색 유입 (`/spinoff/home`) | 플랫폼 내 상영작 라인업 소개 및 고객사 데모 쇼케이스 역할 | S-1 |
| **스핀오프 가이드 모드** | 포스터 링크 및 QR 진입 (`/spinoff/[slug]`) | 상단 뒤로가기만 존재, 하단 탭바 없음. 특정 스핀오프 세계관에 몰입된 상태 | S0, S1, S2, S3, S4 |
| **본전시 안내 모드** (신규) | 온보딩(S1) 또는 아카이브(S3) 링크 진입 (`/exhibition/[main_slug]`) | 본전시 정보 제공 및 예매 전환용 페이지. 예매 완료 시 티켓이 발급되어 본전시 활성화 상태로 전이 | S1.5 |

## 3. 화면 인벤토리 (Inventory)

### S-1. 서비스 홈
* **URL**: `/spinoff/home`
* **주요 정보**: "현재 상영 중인 스핀오프" 라인업 카드 (상설 데모 1개, 파트너 1개 등), Coming Soon 예고 카드
* **이동**: 카드를 누르면 해당 스핀오프의 S0 발견(포스터) 화면으로 이동

### S0. 발견 (디지털 포스터)
* **URL**: `/spinoff/[slug]`
* **주요 정보**: 세계관 키비주얼, 전시명, 전시 한 줄 소개, 입장하기 버튼, 원작 전시 출처 안내
* **이동**: `[입장하기]` 클릭 시 S1 온보딩 화면으로 이동

### S1. 입장 (온보딩)
* **URL**: `/spinoff/[slug]/onboarding`
* **주요 정보**:
  * 인트로 애니메이션 (스킵 가능)
  * 분기 질문: "본전시 〈심해의 환상〉, 보셨나요?"
    * **"아니오, 처음이에요"**: 감정 키워드 칩 2~3개 선택 + **[원작이 궁금하다면? 본전시 정보 보기 ↗]** 링크 노출
    * **"네, 봤어요"**: 기억에 남는 장면 1개 선택 칩
  * 필수 관람 데이터 수집 동의 안내 (로그인 불필요)
* **이동**: 선택 후 `[관람 시작]` 클릭 시 S2 씬 뷰어로 이동. 본전시 정보 링크 클릭 시 S1.5로 이동

### S1.5. 본전시 상세 정보 및 예매
* **URL**: `/exhibition/[main_slug]` (예: `/exhibition/abyss`)
* **주요 정보**: 본전시 키비주얼, 소개 글, 전시 일정/장소 정보, 티켓 예매 버튼
* **이동**: `[본전시 예매하기]` 클릭 시 S1.5.1 예매 결제창으로 이동

### S1.5.1. 본전시 예매/결제 (신규)
* **URL**: `/exhibition/[main_slug]/booking`
* **주요 정보**: 가상 관람 금액, 매수 선택, 예약자 정보 입력(이름/이메일/번호), 가상 결제수단 선택
* **이동**: `[결제하기]` 클릭 시 1.5초 결제 시뮬레이션 후 S1.5.2 예매완료로 이동

### S1.5.2. 본전시 예매완료 (신규)
* **URL**: `/exhibition/[main_slug]/success`
* **주요 정보**: 예매 완료 메시지, 예약 번호, 모바일 pass 바코드/QR, 세션 활성화 및 닉네임 로컬 기록
* **이동**: `[본전시 작품목록 보기]` 클릭 시 `/artwork`(본전시 작품목록)로 이동. `[홈으로]` 클릭 시 메인 홈(`/home`)으로 이동

### S1.6. 본전시 완료 및 초대권 발급 (신규)
* **URL**: `/exhibition/[main_slug]/completed`
* **주요 정보**: 본전시 관람 완료 축하 문구, 수집된 감정 기반의 **스핀오프 특별 초대장 티켓** 비주얼
* **이동**: `[스핀오프 전시 보러가기 ↗]` 클릭 시 `/spinoff/abyss-wave`(S0 발견 포스터)로 이동하여 스핀오프 관람 유도

### S2. 관람 (씬 뷰어)
* **URL**: `/spinoff/[slug]/viewer`
* **주요 정보**:
  * 세로 스와이프 가능한 3~4개의 신규 장면 카드 (이미지 + 사운드 + 짧은 내러티브)
  * AI 세계관 안내자 대화 오버레이 (버블 형태)
  * 사용자가 S1에서 고른 감정/장면에 따라 장면 설명의 수식어가 변하는 개인화 요소 노출
* **이동**: 장면 탐색 완료 후 하단 `[여정 기록하고 아카이브 받기]` 버튼 클릭 시 S3 아카이브 화면으로 이동

### S3. 기념 (나의 아카이브)
* **URL**: `/spinoff/[slug]/archive`
* **주요 정보**:
  * 관람 궤적 기반의 비주얼 아카이브 카드 (감정 컬러 믹스, 머문 장면 모자이크)
  * 사용자가 나눈 AI 대화 발췌문 인용구
  * SNS 공유 카드 만들기 버튼, 출구 굿즈 구매 버튼
  * **[조건부 CTA]** S1 온보딩에서 "아니오"를 선택했던 유저에게만 하단에 **[본전시 〈심해의 환상〉 보러가기 ↗]** 버튼을 강조하여 노출
* **이동**: 본전시 보러가기 클릭 시 S1.5 본전시 상세 화면으로 이동. 여정 종료 시 메인 계정 홈(`/home`)으로 이동

### S4. 여운 (재방문)
* **URL**: `/spinoff/[slug]/retention`
* **주요 정보**:
  * D+2 시나리오 알림 모사 배너
  * 다섯 번째 스핀오프 보너스 장면 오픈 및 대화창 활성화
  * 누적된 감정 궤적 아카이브 재확인 링크

## 4. 유저 상태값 (Session State) 및 로컬스토리지 명세

데이터 레이어 보류 및 격리 원칙에 맞춰 모든 상태는 브라우저 `localStorage`와 React Context에서 공유됩니다.

### 4-1. 스핀오프 세션 객체 (`afterglow_spinoff_session`)
```json
{
  "spinoff_id": "pj_abyss_pop",
  "nickname": "고요한 윤슬",
  "seen_main_exhibition": false,
  "selected_keywords": ["고요함", "압도적"],
  "selected_scene": null,
  "visited_scenes": {
    "scene_1": 45,
    "scene_2": 30
  },
  "chat_history": [
    { "sender": "ai", "text": "빛이 닿지 않는 정원에 오신 것을 환영합니다." },
    { "sender": "user", "text": "왜 이렇게 고요하죠?" }
  ]
}
```

### 4-2. 본전시 세션 객체 (`afterglow_session`)
* S1.5 본전시 상세에서 예매/티켓 코드를 등록하는 경우, 기존 본전시 세션 객체가 활성화(`isPurchased = true` 상태로 판정받기 위한 `afterglow_session` 기록)되어 계정 모드 홈(`/home`)에 "활성화된 관람 티켓" 카드가 노출됩니다.
