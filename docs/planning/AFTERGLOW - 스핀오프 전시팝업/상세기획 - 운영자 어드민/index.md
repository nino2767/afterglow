# 상세기획 - 운영자 어드민

## 1. 방향

AFTERGLOW 운영자 툴은 미디어아트 전시 기획사(B2B)가 전시 IP를 활용해 **스핀오프 팝업을 기획·운영하는 AI 기반 SaaS 어드민 툴**입니다. 기획자가 흰 종이(Blank Page) 앞에 앉지 않도록, AI가 항상 먼저 움직입니다.

- **Zero-Draft**: 운영자는 절대 빈 화면에서 시작하지 않습니다. AI가 0번째 초안을 먼저 제시합니다.
- **근거 선행**: 모든 AI 제안에는 "왜"가 먼저 나옵니다. 관람객 데이터 근거 없는 컨셉 카드는 표시되지 않습니다.
- **B2B↔B2C 루프**: 운영자의 고퀄리티 팝업 → 관람객 경험 → 관람객 데이터 → 다시 운영자 기획 근거로 순환합니다.

| --- | --- | --- |

## 7. Phase별 상세 Flow

**[버전 1: 개별 작품을 꼼꼼히 학습시키는 기술적 Flow]**와 **[버전 2: 스핀오프 전시 기획안을 뽑아내는 목적 지향적 Flow]**를 결합해야, B2B 운영자가 이 시스템을 왜 써야 하는지(스핀오프 기획), 그리고 시스템이 어떻게 작동하는지(AI 학습)가 모두 충족됩니다.

이 두 가지를 완벽하게 믹스한 ** B2B 최우선 Flow**를 정리합니다.

---

## 💼 B2B (운영자) 최우선 Flow: AI 학습부터 스핀오프 기획까지

> **핵심 전략:** 운영자가 전시 데이터를 입력하여 **'B2C용 AI 큐레이터를 학습'**시키는 동시에, 그 데이터를 바탕으로 **'초고속 스핀오프 팝업 기획안'**을 도출해 내는 일석이조의 구조입니다.

---

### Phase 1. 마스터 자산 등록 (기본 세팅)

| 사용자 Task (액션) | 서비스/AI 대응 (Back-end) | 결과물 (B2B 가치) |
| --- | --- | --- |
| 1. **'새 프로젝트 생성'** 버튼 클릭 | 프로젝트 워크스페이스 자동 생성 | 전시 단위 관리 체계 구축 |
| 2. 전시 메인 포스터 및 전체 기획서(PDF) 업로드 | Asset Analyzer 가동 (전체 세계관, 핵심 테마 및 톤앤매너 추출) | 프로젝트 기초 환경 구성 |
| 3. **전시 공간 구조 입력** (섹션/구간 설정, 동선 배치) | 공간 맵핑 엔진이 구간별 데이터 수집 포인트 자동 세팅 | 구간별 체류·이탈 측정 기반 마련 |
| 4. **작품 배치 정보 등록** (작품별 위치, QR 코드 자동 생성) | 작품-공간 매핑 DB 구축, QR 코드 일괄 생성 및 배포용 PDF 출력 | 현장 설치용 QR 에셋 확보 |

> 💡 **연동 모듈:** 전시 반응 분석 모듈(상위기획서 7-1)의 데이터 수집 인프라가 이 Phase에서 세팅됩니다.

---

### Phase 2. 개별 작품 학습 ⭐️ (B2C 대응용)

| 사용자 Task (액션) | 서비스/AI 대응 (Back-end) | 결과물 (B2B 가치) |
| --- | --- | --- |
| 1. 개별 작품 사진/영상 에셋 업로드 | Vision LLM 이미지 인식 매핑 | 작품별 AI 인식 모델 구축 |
| 2. 도슨트 스크립트 및 작가 의도 입력 | RAG 세팅 (질의응답 시나리오 구축) | 할루시네이션 없는 정확한 B2C AI 도슨트 완성 |
| 3. **'AI 지식베이스 학습' 클릭** | 작품별 Knowledge Base 벡터 DB 생성 | B2C 관람객 대응 AI 큐레이터 가동 준비 완료 |
| 4. **감정 태그 후보 세팅** (작품별 예상 감정 키워드 사전 등록) | 감정 태그 추천 엔진 초기값 설정 | B2C 앱의 감정 태그 Chip UI에 반영 |

> 💡 **B2C 연동 포인트:** 이 Phase에서 학습된 데이터가 B2C 관람객 Flow의 Level 1(스냅 & 정보 카드)과 Level 2(AI 큐레이터 채팅)의 응답 품질을 결정합니다.

---

### Phase 3. 데이터 수집 & 반응 패턴 분석 ⭐️ (신규 추가)

| 사용자 Task (액션) | 서비스/AI 대응 (Back-end) | 결과물 (B2B 가치) |
| --- | --- | --- |
| 1. 전시 오픈 후 **실시간 데이터 수집 현황** 탭 확인 | B2C 앱에서 유입되는 데이터 실시간 집계 (QR 스캔, 체류시간, 촬영, 감정 태그, 동선, 이탈 구간) | 전시 반응 데이터 실시간 모니터링 |
| 2. **작품/구간별 반응 히트맵** 확인 | 반응 패턴 분석 모듈 가동 (몰입 구간, 반복 관심 요소, 이탈 구간 자동 분석) | 어떤 작품에서 몰입이 높고 어디서 이탈이 발생하는지 시각화 |
| 3. **반응 키워드 랭킹** 확인 (작가/서사/감정/매체 축) | 반응 키워드 추출 모듈 가동 | 관람객이 가장 많이 반응한 축(감정, 작가, 서사 등) 파악 |
| 4. **세그먼트별 분석** 확인 (연령대, 선호군, 반응 유형별) | 세그먼트별 추천 모듈 가동 (20대 여성 / 미디어아트 선호군 / 스토리 반응군 등) | 타깃별 차별화된 후속 기획 근거 확보 |

> 💡 **연동 모듈:** 상위기획서 7-1) 전시 반응 분석, 7-2) 반응 키워드 추출, 7-4) 세그먼트별 추천 모듈이 이 Phase에서 작동합니다.

### Phase 4. 스핀오프 기획 도출 ⭐️ (수익화 용도)

| 사용자 Task (액션) | 서비스/AI 대응 (Back-end) | 결과물 (B2B 가치) |
| --- | --- | --- |
| 1. **'스핀오프 자동 기획' 버튼 클릭** | Concept Bot 가동 (축적된 반응 데이터 기반 컨셉 자동 도출) | 데이터 기반 스핀오프 컨셉 후보 생성 |
| 2. AI가 제안한 **6가지 유형별 컨셉 카드** 비교·검토 | 유형별 컨셉 카드 제시: ① 작가 확장형 ② 서사 확장형 ③ 감정/분위기 확장형 ④ 매체 확장형 ⑤ 테마 재구성형 ⑥ 개인화 추천형 — 각 카드에 **관람객 욕구 + 가치 훅 + Bridge 트리거 문구** 포함 (하단 상세 참조) | 운영자가 "이 컨셉으로 관람객이 온다"는 확신을 갖고 최적 컨셉 선택 가능 |
| 3. 컨셉 1~2종 선택 후 **상세 기획안 생성 요청** | 선택된 컨셉 기반 상세 시나리오 + 매칭 커스텀 MD(굿즈) 리스트 자동 생성 | 24시간 내 도출되는 스핀오프 전시/팝업 운영 가이드북 |
| 4. **제안서 초안 자동 생성** (보고용) | 제안서 초안 생성 모듈 가동 (제안 제목, 배경, 근거 데이터, 컨셉안 자동 구성) | 사내 보고 및 파트너 공유용 제안서 즉시 출력 |
| 5. **컨셉 카드 추가** (3가지 경로) | ① **운영자 직접 추가**: "+ 커스텀 컨셉" 버튼 → 빈 카드에 타이틀·방향성 입력 시 AI가 추천 MD 5종 + Bridge 트리거 문구 자동 완성 제안 / ② **AI 데이터 기반 자동 추가**: 관람 데이터에서 기존 6종에 없는 새 반응 패턴 감지 시 "새 컨셉 감지됨" 알림 → 운영자 승인 후 신규 카드 추가 / ③ **운영 중 컨셉 진화**: 팝업 운영 중 유입되는 신규 데이터로 기존 카드 자동 정교화 or 신규 카드 추가 제안 | 컨셉 카드가 고정 제안물이 아닌, 관람 데이터 → 컨셉 → 관람객 재유입의 **살아있는 루프 장치**가 됨 |

> 📌 **6가지 컨셉 유형 × B2C 가치 레이어** — 각 컨셉 카드에 포함되는 관람객 가치 필드

| 컨셉 유형 | 관람객 욕구 (Visitor Desire) | 관람객 가치 훅 (Value Hook) | Bridge 트리거 문구 예시 |
| --- | --- | --- | --- |
| ① 작가 확장형 | 전시에서 만난 작가 세계에 더 깊이 들어가고 싶다 | 전시장 밖에서 작가의 작업실을 직접 경험하는 시간 | "S님이 가장 오래 머문 작품 — 그 작가가 직접 설계한 공간이 있어요." |
| ② 서사 확장형 | 전시의 이야기가 끝나지 않았으면 좋겠다 | 전시에서 미완으로 끝난 이야기를 내가 직접 완성하는 체험 | "오늘 탐험한 심연의 끝은 아직 열려있어요. 팝업에서 결말을 직접 써보세요." |
| ③ 감정/분위기 확장형 | 전시에서 느낀 감정을 오래 붙잡아두고 싶다 | 전시가 남긴 감정을 오감으로 소장할 수 있는 공간 | "'고요한'이라는 단어를 3번 말씀하셨어요. 그 감각을 향기로 담아드릴게요." |
| ④ 매체 확장형 | 미디어아트 기술이 신기했다, 직접 만져보고 싶다 | 관람객에서 창작자로 — 미디어아트 기술을 직접 조작하는 체험 | "S님이 촬영한 빛의 굴절, AI가 당신만의 작품으로 렌더링해드릴게요." |
| ⑤ 테마 재구성형 | 같은 주제를 다른 감각으로 경험해보고 싶다 | 눈으로 본 것을 미각·청각·촉각으로 다시 경험 | "오늘 눈으로 느낀 심해의 온도 — 이번엔 혀끝으로 경험해보세요." |
| ⑥ 개인화 추천형 | 나만을 위한 무언가를 갖고 싶다 | 내 관람 데이터로 만들어진 세상에 하나뿐인 굿즈와 공간 | "오늘 S님의 관람 여정으로 만들어진 단 하나의 아카이브가 준비됐어요." |

> 💡 **연동 모듈:** 상위기획서 7-3) 스핀오프 컨셉 추천, 7-5) 제안서 초안 생성 모듈이 이 Phase에서 작동합니다. 컨셉 카드 자동 추가(Task 5-②)는 7-2) 반응 키워드 추출 모듈과 연동됩니다.

### Phase 5. 라이브 대시보드 (데이터 성과)

| 사용자 Task (액션) | 서비스/AI 대응 (Back-end) | 결과물 (B2B 가치) |
| --- | --- | --- |
| 1. 전시 기간 중 **실시간 현황 탭** 확인 | Insight Reporter (관람객 앱 내 행동 데이터 실시간 시각화) | 전시 운영 중 즉각적 의사결정 지원 |
| 2. 관람객 **반응 키워드 랭킹** 확인 | 시간대별·일별 트렌드 분석 | 전시 기간 중 운영 최적화 근거 |
| 3. **초대장 클릭률(CTR) 및 팝업 방문율** 확인 | 스핀오프 전환 퍼널 분석 | 스핀오프 연결 효과 실시간 측정 |
| 4. **성과 증빙 리포트 출력** | 보고용 요약 자료 자동 생성 (PDF/PPT) | 차기 전시를 위한 정밀 취향 분석 및 성과 증빙 리포트 |

### Phase 6. 포스트 전시 운영 (신규 추가)

| 사용자 Task (액션) | 서비스/AI 대응 (Back-end) | 결과물 (B2B 가치) |
| --- | --- | --- |
| 1. 전시 종료 후 **최종 데이터 자산화 리포트** 확인 | 데이터 자산화 리포트 모듈 가동 (전시별 데이터 축적 현황, 유사 전시 비교) | AFTERGLOW 전시 반응 DB에 해당 전시 데이터 영구 축적 |
| 2. **후속 반응 데이터 모니터링** (초대장 발송 후 클릭, 열람, 전환 추적) | 후속 반응 트래커 가동 | 스핀오프 전환율 최종 측정 및 ROI 산출 |
| 3. **유사 전시 비교 분석** 요청 | 누적 DB 기반 장르·작가군·시즌별 반응 패턴 비교 | 향후 스핀오프 성공 가능성 예측 고도화 |
| 4. **차기 전시 기획 인사이트** 확인 | 전시 간 데이터 크로스 분석 및 차기 제안 자동 생성 | 데이터 기반 차기 전시 기획 방향성 확보 |

> 💡 **연동 모듈:** 상위기획서 7-6) 데이터 자산화 리포트 모듈이 이 Phase에서 작동합니다. 전시별 데이터가 누적될수록 추천 정교도와 산업 인사이트가 강화됩니다.

## 💡 기획 완성의 마침표

운영자의 전체 행동 흐름이 **[1. 자산 등록 → 2. AI 학습 → 3. 데이터 수집 & 분석 → 4. 스핀오프 기획 도출 → 5. 라이브 대시보드 → 6. 포스트 전시 운영]** 이라는 6단계의 완결된 SaaS 어드민 구조를 갖추게 되었습니다.

### 기존 대비 보강된 핵심 포인트

- **Phase 1 보강:** 전시 공간 구조 입력, 작품 배치, QR 코드 생성 등 현장 세팅 Task 추가
- **Phase 3 신규:** B2C에서 유입되는 반응 데이터 수집·분석 전용 Phase 분리 (세그먼트 분석 포함)
- **Phase 4 보강:** 6가지 스핀오프 유형별 컨셉 비교 구조 + 제안서 자동 생성 Task 추가
- **Phase 5 보강:** 초대장 CTR, 팝업 방문율, 성과 증빙 리포트 출력 Task 추가
- **Phase 6 신규:** 전시 종료 후 데이터 자산화, 후속 반응 추적, 유사 전시 비교 Phase 추가
- **전체 보강:** 각 Phase별 상위기획서 자동화 모듈(7개)과의 매핑 명시, B2C↔B2B 데이터 연동 포인트 기술
