# 상세기획 - 고객사 관리 어드민

## 1. 방향

AFTERGLOW 어드민은 **3종 어드민 구조**로 분리됩니다.

| 어드민 | 사용자 | 핵심 역할 |
| --- | --- | --- |
| **고객사 관리 어드민** | AFTERGLOW 자사 직원 | 고객사 등록·계약·플랜·RS 정산 관리 |
| **운영자 어드민 (자사직원용)** | AFTERGLOW 운영팀 | 전체 고객사 전시 운영 총괄, 내부 기능 전체 접근 |
| **운영자 어드민 (고객사용)** | 기획사 현장 운영직원 | 자사 전시만 Phase 1~6 수행, 컨셉 카드 검토·승인 |

> 📌 **운영자 어드민 2종은 MVP 단계에서 하나의 시스템 + Role 분리 방식으로 운영.** 고객사 수가 확대되는 v2.0 이후 고객사용 화이트라벨 어드민으로 분리 검토. → 상세 의사결정 내용은 별도 페이지 참조.

- **접근 격리 원칙**: 고객사 운영직원은 자사 데이터만 열람 가능. 타 고객사 데이터 접근 불가.
- **위임 구조**: 슈퍼 어드민이 고객사를 생성하면, 고객사 어드민이 운영자 계정을 직접 생성·관리.
- **성과 투명성**: 고객사는 자사 전시의 성과 리포트와 RS 정산 내역을 실시간 열람 가능.

## 2. 목표 및 기대효과

| 구분 | 목표 | 기대효과 |
| --- | --- | --- |
| **계정 관리** | 고객사별 독립 어드민 환경 제공 | 데이터 보안 확보 / 고객사 자율 운영 가능 |
| **계약 관리** | 플랜·RS 비율·계약 기간 중앙 관리 | 정산 자동화 / 분쟁 예방 |
| **성과 열람** | 고객사가 자사 전시 성과를 실시간 확인 | 재계약 근거 확보 / 신뢰도 향상 |
| **운영자 관리** | 고객사가 담당 운영자 계정을 직접 생성·권한 부여 | 슈퍼 어드민 병목 제거 |

## 4. 고객사 데이터 항목 (CLIENT 엔티티)

| 필드명 | 타입 | 설명 | 필수 |
| --- | --- | --- | --- |
| **client_id** | PK / auto | 고객사 고유 ID | 자동 |
| **company_name** | string | 고객사명 (예: 피플리) | 필수 |
| **company_site** | url | 공식 웹사이트 | 필수 |
| **contact_name** | string | 담당자명 | 필수 |
| **contact_email** | string | 담당자 이메일 (로그인 ID) | 필수 |
| **contact_phone** | string | 담당자 연락처 | 필수 |
| **plan_type** | enum | basic / pro / enterprise | 필수 |
| **rs_ratio** | float | Revenue Share 비율 (%) | 필수 |
| **contract_start** | date | 계약 시작일 | 필수 |
| **contract_end** | date | 계약 종료일 | 필수 |
| **status** | enum | active / inactive / pending | 필수 |
| **created_at** | datetime | 등록일 (자동) | 자동 |

> 📌 슈퍼 어드민만 생성·수정 가능. 고객사 어드민은 열람만 가능.

## 5. 고객사 어드민 ↔ 운영자 어드민 데이터 연결

![image]()





| 연결 | 방향 | 설명 |
| --- | --- | --- |
| CLIENT → OPERATOR_ACCOUNT | 1:N | 고객사 어드민이 운영자 계정 생성·역할(admin/viewer) 부여 |
| CLIENT → EXHIBITION | 1:N | 고객사 어드민이 전시 등록 및 IP 자산 업로드 |
| OPERATOR_ACCOUNT → EXHIBITION | M:N (PERMISSION) | 운영자는 허가된 전시만 접근 가능 (접근 레벨: full/readonly) |
| EXHIBITION → PERFORMANCE_REPORT | 1:N | 전시 종료 후 성과 리포트 자동 생성 → 고객사 열람 |
| PERFORMANCE_REPORT → CLIENT | N:1 | RS 정산 금액 자동 계산 → 고객사 정산 내역 제공 |
