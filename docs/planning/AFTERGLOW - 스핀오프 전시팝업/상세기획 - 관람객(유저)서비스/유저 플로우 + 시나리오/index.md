# 유저 플로우 + 시나리오

## 1. 관람객 경험 방향

AFTERGLOW의 관람객 경험은 전시 현장에서 끝나지 않는다.

관람객은 전시를 보며 작품을 저장하고, 감정 태그를 남기고, 더 알고 싶은 설명을 확인한다.

이 행동은 관람 종료 후 **나만의 관람 리포트**로 정리되고, 다시 **내가 반응한 작품·감정·주제와 연결된 스핀오프 전시, 콘텐츠, 팝업, 굿즈 추천**으로 이어진다.

즉, AFTERGLOW는 관람객에게 다음과 같은 경험을 제공한다.

| 단계 (총5단계) | 관람객 경험 | 서비스 역할 |
| --- | --- | --- |
| 관람 시작 | 전시 관람 모드 진입 | 관람 세션 생성 |
| 관람 중 | 작품 저장, 감정 선택, 상세 설명 확인 | **반응 데이터 수집** |
| 관람 완료 | 나의 관람 기록 확인 | 관람 리포트 생성 |
| 관람 이후 | 내 감상과 연결된 후속 콘텐츠 확인 | 스핀오프 추천 |
| 재방문/전환 | 알림 신청, 관심 등록, 예매, 구매 | 후속 경험 연결 |

```plain text
↓
관람 방법 안내 확인
↓
작품·섹션 관람
↓

↓
작품 상세 설명 확인

↓

↓

```

# 3. 단계별 유저 플로우 상세

## 3-1. 전시 입장 및 관람 모드 시작

### 관람객 행동

관람객은 전시장 입구, 티켓 확인 지점, 리플렛, 전시 안내 배너 등에 노출된 QR을 스캔한다.

### 화면 흐름

```plain text
입구 QR 스캔
↓
전시 전용 모바일 웹 진입
↓
전시명 확인
↓
관람 모드 시작
↓
데이터 활용 동의
↓
익명 관람 세션 생성
↓
관람 방법 안내 확인
```

### 화면 구성

| 화면 영역 | 내용 |
| --- | --- |
| 전시명 | 현재 관람하는 전시명 |
| 안내 문구 | 전시를 기록하고 리포트를 받을 수 있다는 메시지 |
| 참여 방법 | 작품 QR 스캔, 저장, 감정 태그 선택 안내 |
| 제공 혜택 | 관람 후 나만의 리포트 제공 |
| 동의 영역 | 익명 데이터 수집 및 선택적 알림 동의 |
| CTA | 관람 시작하기 |

### 문구 예시

> 오늘 본 작품과 감정을 기록해보세요.

> 관람이 끝나면 나만의 전시 리포트를 확인할 수 있어요.

### 수집 데이터

| 데이터 | 설명 |
| --- | --- |
| 전시 ID | 어떤 전시에서 발생한 세션인지 식별 |
| 관람 세션 ID | 익명 또는 회원 기반 세션 |
| 관람 시작 시간 | 입장 시점 |
| 진입 경로 | 입구 QR, 티켓 QR, 리플렛 QR 등 |
| 데이터 동의 여부 | 수집 및 활용 동의 상태 |

## 3-2. 관람 준비 및 관심 키워드 선택

### 관람객 행동

관람객은 관람 전 간단한 안내를 확인하고, 선택적으로 관심 키워드를 고른다.

### 화면 흐름

```plain text
관람 모드 시작
↓
관람 안내 확인
↓
선택 사항: 관심 키워드 선택
↓
관람 시작
```

### 관심 키워드 예시

| 유형 | 선택지 |
| --- | --- |
| 관심 요소 | 작가 이야기, 작품 배경, 색감, 시대 배경, 공간 연출 |
| 감상 방식 | 깊게 보고 싶어요, 가볍게 둘러보고 싶어요 |
| 후속 관심 | 관련 전시, 굿즈, 강연, 온라인 콘텐츠, 팝업 |

### 기획 포인트

관심 키워드 선택은 필수가 아니라 선택 사항으로 제공한다.

입장 직후부터 입력 부담이 커지면 관람 모드 진입률이 낮아질 수 있으므로 **건너뛰기**가 가능해야 한다.

## 3-3. 작품 관람 및 작품별 반응 수집

### 관람객 행동

관람객은 작품을 감상하다가 더 알고 싶은 작품의 QR을 스캔한다.

### 화면 흐름

```plain text
작품 앞 QR 스캔
↓
작품 상세 화면 진입
↓
작품 기본 정보 확인
↓
1분 해설 확인
↓
더 깊게 보기 선택
↓
감정 태그 선택
↓
작품 저장
↓
관련 작품 또는 다음 작품 이동
```

### 작품 상세 화면 구성

| 화면 영역 | 내용 |
| --- | --- |
| 작품 이미지 | 작품 대표 이미지 |
| 작품 정보 | 작품명, 작가명, 제작 연도, 매체 |
| 한 줄 소개 | 작품을 빠르게 이해할 수 있는 요약 |
| 1분 해설 | 현장에서 부담 없이 읽을 수 있는 짧은 설명 |
| 더 깊게 보기 | 작가 서사, 시대 배경, 관계성, 제작 비화 |
| 감정 태그 | 이 작품을 보고 느낀 감정 선택 |
| 저장하기 | 나의 리포트에 저장 |
| 공유하기 | 작품 링크 또는 감상 카드 공유 |
| 관련 작품 | 같은 작가, 주제, 감정의 다른 작품 |

### 감정 태그 예시

- 따뜻했어요
- 쓸쓸했어요
- 압도적이었어요
- 더 알고 싶어요
- 다시 보고 싶어요
- 사진으로 남기고 싶어요
### 수집 데이터

| 관람객 행동 | 수집 데이터 | 해석 |
| --- | --- | --- |
| 작품 QR 스캔 | 작품별 스캔 로그 | 명시적 관심 |
| 1분 해설 확인 | 기본 설명 열람 | 기본 탐색 |
| 더 깊게 보기 클릭 | 상세 콘텐츠 클릭 | 서사 관심 |
| 작품 저장 | 저장 로그 | 지속 관심 |
| 감정 태그 선택 | 감정 반응 | 정서적 몰입 |
| 공유하기 | 공유 로그 | 확산 가능성 |
| 관련 작품 클릭 | 연관 콘텐츠 클릭 | 확장 탐색 |

## 3-4. 섹션 관람 및 주제 반응 수집

### 관람객 행동

관람객은 섹션 입구 또는 출구 QR을 통해 해당 구간의 감상 포인트를 확인하고, 가장 인상 깊었던 요소를 선택한다.

### 화면 흐름

```plain text
섹션 QR 스캔
↓
섹션 소개 확인
↓
섹션 내 주요 작품 확인
↓
가장 인상 깊은 요소 선택
↓
섹션 감정/관심 키워드 선택
↓
다음 섹션 이동
```

### 섹션 상세 화면 구성

| 화면 영역 | 내용 |
| --- | --- |
| 섹션명 | 예: 고갱과의 관계, 아를의 시간 |
| 감상 포인트 | 해당 구간에서 주목할 요소 |
| 주요 작품 | 섹션 내 대표 작품 리스트 |
| 감상 질문 | 이 구간에서 가장 인상 깊었던 점 |
| 키워드 선택 | 관계성, 색감, 시대 배경, 공간 연출 등 |
| 다음 섹션 안내 | 관람 흐름 유지 |

### 질문 예시

> 이 구간에서 가장 인상 깊었던 점은 무엇인가요?

선택지 예시:

- 작가의 이야기
- 인물 간 관계
- 작품의 색감
- 시대적 배경
- 공간 분위기
- 잘 모르겠어요
### 수집 데이터

| 관람객 행동 | 수집 데이터 | 해석 |
| --- | --- | --- |
| 섹션 QR 스캔 | 섹션 진입 로그 | 구간 관심 |
| 주요 작품 클릭 | 섹션 내 작품 관심 | 세부 관심 |
| 키워드 선택 | 섹션별 관심 키워드 | 주제 반응 |
| 다음 섹션 이동 | 동선 데이터 | 이탈 여부 판단 |

## 3-7. 관람 리포트 확인

### 관람객 행동

관람객은 자신의 관람 결과를 리포트 형태로 확인한다.

### 화면 흐름

```plain text
관람 리포트 진입
↓
관람 요약 확인
↓
저장한 작품 확인
↓
감정 키워드 확인
↓
관심 주제 확인
↓
나의 관람 성향 확인
↓
이어볼 콘텐츠 확인
```

### 리포트 화면 구성

| 영역 | 내용 |
| --- | --- |
| 나의 관람 요약 | 저장 작품 수, 감정 태그, 관심 섹션 |
| 가장 인상 깊었던 작품 | 저장/상세 보기/체류 기반 작품 |
| 나의 감정 키워드 | 쓸쓸함, 위로, 더 알고 싶음 등 |
| 관심 주제 | 작가 관계성, 편지, 색감, 공간 연출 등 |
| 관람 성향 요약 | 서사 탐색형, 이미지 소장형, 감정 몰입형 등 |
| 이어볼 콘텐츠 | 아티클, 오디오, 영상, 관련 작품 |
| 스핀오프 추천 | 후속 전시, 팝업, 굿즈, 강연 등 |

### 리포트 문구 예시

> 이번 전시에서 당신은

> 작품의 색감보다 **작가의 관계성과 감정 서사**에 더 깊게 반응했어요.

> 저장한 작품과 선택한 감정 태그를 보면

> **편지, 고독, 관계성**과 연결된 콘텐츠에 관심이 높았어요.

## 3-8. 관람 이후 이어볼 콘텐츠 추천

### 관람객 행동

관람객은 리포트 하단에서 자신의 감상과 연결된 짧은 콘텐츠를 확인한다.

### 화면 흐름

```plain text
관람 리포트 확인
↓
내 반응 키워드 기반 콘텐츠 추천
↓
아티클 / 오디오 / 영상 콘텐츠 확인
↓
더 깊은 이야기 보기
↓
스핀오프 관심 등록 유도
```

### 추천 콘텐츠 예시

| 반응 키워드 | 이어볼 콘텐츠 |
| --- | --- |
| 관계성 | 고흐와 고갱이 함께한 시간 |
| 편지 | 편지로 남겨진 작가의 내면 |
| 고독 | 작품 속 고독과 위로의 감정 |
| 색감 | 노란색이 작품에 반복된 이유 |
| 공간 연출 | 몰입형 미디어아트 제작 과정 |

### CTA 예시

- 이어서 보기
- 이 이야기 더 보기
- 관련 콘텐츠 저장하기
- 이 주제로 전시가 열린다면 알려주세요
### 기획 포인트

관람 직후에는 바로 예매를 유도하기보다, **짧은 콘텐츠를 먼저 제공한 뒤 스핀오프 관심 등록으로 연결**하는 흐름이 자연스럽다.

관람객 입장에서는 판매 제안이 아니라, 자신의 감상을 더 확장하는 경험으로 받아들일 수 있다.

## 3-9. 스핀오프 추천 및 관심 등록

```plain text

↓
추천 이유 확인
↓
관심 있는 주제 선택
↓
관심 등록 / 오픈 알림 신청
↓
알림 수신 동의
```

## 3-10. 스핀오프 사전예약 및 예매 연결

### 관람객 행동

스핀오프 전시나 팝업이 이미 오픈 예정인 경우, 관람객은 사전예약 또는 예매로 이동한다.

### 화면 흐름

```plain text
스핀오프 추천 카드 선택
↓
스핀오프 상세 페이지 진입
↓
전시/팝업 컨셉 확인
↓
내 관람 데이터와의 연결 이유 확인
↓
일정 및 장소 확인
↓
사전예약 / 예매하기 선택
↓
예약 완료
↓
오픈 전 리마인드 수신
```
