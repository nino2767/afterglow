# CLAUDE.md

## 작업 규칙

### 문서·자료 확인 순서
1. **로컬 폴더/레포를 항상 먼저 확인한다** (이 afterglow 레포 및 연결된 폴더).
2. 로컬에 없을 때만 노션을 확인하되, **노션 확인을 진행하기 전에 먼저 사용자에게 물어보고 승인받는다.**
3. 승인 없이 노션을 검색·조회하지 않는다.

### 토큰 절약
4. **넓은 검색 대신 경로 특정 후 부분 읽기.** 노션 검색·레포 전체 grep은 최후 수단. 관제탑(진행현황 로그) 또는 repo `docs/` 구조를 먼저 보고 필요한 파일만 읽는다.
5. **긴 파일·페이지는 통째로 읽지 않는다.** 헤더/목차 먼저 훑고 필요한 섹션만 offset으로 읽는다. (짧은 문서는 전체 OK)
6. **답변에 원문 장문을 재인용하지 않는다.** 근거는 `파일:라인` 또는 섹션명으로 가리키고 내용을 그대로 복붙하지 않는다.
7. **산출물 형식을 매번 되묻지 않는다.** 문서 보강은 기존 파일 부분 edit이 기본. 새 형식(docx 등)은 명시적으로 요청할 때만.
